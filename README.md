Pokémon Defensive Profile Calculator
This project evaluates a team of up to six Pokémon and calculates their combined defensive coverage against every type in the game. 

How to Use
Run via Web Browser: Click the link to select your generation, pick your team's types from the dropdown menus, and generate a visual breakdown.

Run Locally via Terminal: Originally built as a command-line tool, you can clone the repository and use the provided Makefile to compile the source (.cpp) files. Once compiled, you can execute the program directly in your terminal to calculate profiles using standard CLI text inputs or by uploading a txt file formatted like template.txt
